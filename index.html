CodX Editor Starter

Build and preview web apps instantly

Write HTML, CSS, and JavaScript in separate files and see updates live in the preview panel.

What you can do

  • Create and switch between multiple files
  • Use Auto-Run for instant preview updates
  • Debug quickly with the built-in console
  • Import/Export projects as ZIP files

Controls

  • Ctrl/Cmd + S Export your project as a ZIP
  • Ctrl/Cmd + Enter Run preview manually
  • Ctrl/Cmd + Q Create a new file
  • Ctrl/Cmd + Shift + C Toggle console
  • Esc Exit Zen Mode
  • Type cxstart in an empty HTML file and press Enter
  • Ctrl/Cmd + C, then X Open developer tools
